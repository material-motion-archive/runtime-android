# 6.0.0

## Breaking changes

* Performer.getTarget() now returns Object rather than implicit cast.
* Deprecated Runtime removed.

## New deprecations

## New features

* Plans and Performers are now type safe. Generics are used to define the type of target a plan and performer can act on. Motion library authors should audit their plans and performers to enable generics.

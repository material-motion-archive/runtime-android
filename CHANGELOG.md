# 2.0.0

## New features

* Composable performing. Performers can now emit plans.
* Continuous performing. These APIs were previously named Delegated performing.
* Deprecated `DelegatedPerformance`.

## Breaking changes

* Removed previously deprecated non-tokenized symbols in `DelegatedPerformance`.
